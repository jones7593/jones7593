Three types of lists used to code HTML web pages.

Lists Usage
Unordered List An unordered list is used to display items that do not have to be arranged in any particular order, such as a shopping list. The tag <ul> is used to start the list.
Ordered List An ordered list is used to display items that do have to be arranged in any particular order, such as a recipes or instructions. The tag <ol> is used to start the list.
Definition List An definition list is used to display a term or word and it's definition or meaning, like the glossary I created below. The tag <dl> is used to start the list.

Terms used during web development.

HTML
Hyper Text Markup Language is the standard markup language used to code HTML documents.
CSS
Cascading Style Sheets is considered a presentational language that is used to format HTML documents with fonts, color and spacing.
FTP
File Transfer Protocol is the standard communication protocol used to upload or download files to and from computers and servers over the Internet.
SSH
Secure Shell Socket is a protocol that allows users to establish secure connections to remote servers.
